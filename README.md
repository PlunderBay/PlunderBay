# PlunderBay

##Running the project

###Running the server
*Open server directory
```cd server```
*Install dependencies
```npm install```
*Run server
```npm run dev```

###Running the client
*Start a new terminal
*Open client directory 
```cd client```
*Install dependencies
```npm install```
*Run client
```npm run dev```
*Open [The client](http://localhost:8080/) in your browser
